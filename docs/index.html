Flexi

A grid system based on the flex display property.

GitHub

Responsive

Responsive modifiers enable specifying different column sizes, offsets, alignment and distribution at xs, sm, md & lg viewport widths.

Child columns class examples:
col-[#]:   omit the size namespace for extra small and up*
col-sm-[#]:   append the -sm namespace for small and up*
col-md-[#]:   append the -md namespace for medium and up*
col-lg-[#]:   append the -lg namespace for large and up* 
<div class="row">
    <div class="col-12
                col-sm-8
                col-md-6
                col-lg-4">
        <div class="box">Responsive</div>
    </div>
</div>

Fluid

Percent based widths allow fluid resizing of columns and rows.

.col-6 {
    flex-basis: 50%;
}

Simple Syntax

All you need to remember is row, column, content.

<div class="row">
    <div class="col-12">
        <div class="box">12</div>
    </div>
</div>

Offsets

Offset a column

<div class="row">
    <div class="col-offset-3 col-9">
        <div class="box">offset</div>
    </div>
</div>

Auto Width

Add any number of auto sizing columns to a row. Let the grid figure it out.

<div class="row">
    <div class="col">
        <div class="box">auto</div>
    </div>
</div>

Shrink Width

Add any number of shrinked sizing columns to a row. Let the grid figure it out.

<div class="row">
    <div class="col shrink auto-md">
        <div class="box">shrink</div>
    </div>
</div>

Nested Grids

Nest grids inside grids inside grids.

<div class="row collapse">
    <div class="col">
        <div class="box">
            <div class="row">
                <div class="col">
                    <div class="box">auto</div>
                </div>
            </div>
        </div>
    </div>
</div>

Alignment

Add classes to align elements to the start or end of a row as well as the top, bottom, or center of a column

.start or .start-[sm|md|lg] 

<div class="row start">
    <div class="col-6">
        <div class="box">
            start
        </div>
    </div>
</div>

.center or .center-[sm|md|lg] 

<div class="row center">
    <div class="col-6">
        <div class="box">
            start
        </div>
    </div>
</div>

.end or .end-[sm|md|lg] 

<div class="row end">
    <div class="col-6">
        <div class="box">
            end
        </div>
    </div>
</div>
